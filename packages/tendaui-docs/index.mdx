## 📋 组件列表

### 基础组件

- [Button 按钮](/?path=/docs/components-button--docs) - 触发操作的按钮组件
- Icon 图标 - 语义化的图标组件

### 数据录入

- [Input 输入框](/?path=/docs/组件-input-输入框--docs) - 通过鼠标或键盘输入内容
- Checkbox 复选框 - 在一组可选项中进行多选
- Switch 开关 - 开关选择器
- Select 选择器 - 下拉选择器
- [Form 表单](/?path=/docs/组件-form-表单--docs) - 表单数据收集、校验和提交
- TagInput 标签输入 - 标签输入组件

### 数据展示

- Badge 徽标 - 图标右上角的圆形徽标数字
- Tag 标签 - 进行标记和分类的小标签
- Loading 加载 - 加载状态展示

### 反馈

- Alert 警告提示 - 警告提示，展现需要关注的信息
- Dialog 对话框 - 在当前页面打开一个浮层
- Notification 通知 - 全局展示通知提醒信息

### 其他

- Popup 弹出层 - 弹出式的气泡/浮层
- Portal 传送门 - 将子节点渲染到 DOM 树的另一个位置
- ConfigProvider 全局配置 - 全局配置组件

## 🎯 文档特点

### 📖 详细的 API 文档

每个组件都有完整的 API 文档，包括：

- Props 属性说明
- Events 事件回调
- Methods 实例方法
- CSS Variables 样式变量

### 💻 丰富的代码示例

提供多种场景的代码示例：

- 基础用法
- 不同状态
- 样式变体
- 进阶用法
- 最佳实践

### 🎨 可交互的演示

所有示例都可以实时编辑和预览，帮助你快速理解组件用法。

### 📱 响应式设计

文档支持在不同设备上查看，可以通过工具栏切换视口大小。

### 🌗 主题切换

支持亮色/暗色主题切换，测试组件在不同主题下的表现。

## 💡 使用建议

### 对于新手

1. **先读介绍**：了解 TendaUI 的设计理念和特点
2. **快速开始**：跟随教程搭建第一个应用
3. **浏览组件**：查看组件列表，了解可用组件
4. **动手实践**：复制示例代码到项目中尝试

### 对于进阶用户

1. **设计规范**：深入了解设计系统
2. **最佳实践**：学习性能优化和架构设计
3. **API 文档**：查阅详细的 API 说明
4. **贡献代码**：参与开源项目开发
