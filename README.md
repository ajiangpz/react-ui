# TWUI React

一个基于 Tailwind CSS 构建的轻量级 React UI 组件库。

## 安装

```bash
npm install twui-react
# 或
yarn add twui-react
# 或
pnpm add twui-react
```

## 使用

```jsx
import { Button } from 'twui-react';

function App() {
  return (
    <Button>点击我</Button>
  );
}
```

## 特性

- 🎨 精心设计的现代化组件
- ⚡️ 基于 Tailwind CSS 构建
- 🚀 使用 TypeScript 开发
- 🎯 支持按需加载
- 📦 体积轻量
- 🎭 高度可定制的主题
